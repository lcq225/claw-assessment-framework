# ClawAF - Claw评价框架

> 用于评价基于Claw体系的AI Agent系统智慧能力的标准化框架

---

## 概述

**ClawAF (Claw Assessment Framework)** 是一套标准化评价框架，用于评估基于Claw体系的AI Agent系统的智慧能力，包括 OpenClaw、CoPaw、qclaw、winClaw、zeroclaw 等所有变种。

### 目标

提供通用的AI Agent能力测量标准，支持：
- 自我评估
- 团队基准测试
- 进化路线规划
- 行业水平对比

---

## 快速开始

### 安装

```bash
git clone https://github.com/lcq225/claw-assessment-framework.git
cd claw-assessment-framework
pip install -r requirements.txt
```

### 自我评估

```bash
python assess.py
```

### 团队对比

```bash
python compare.py --users user1,user2,user3
```

## 8大评价维度

### 1. 身份认知度

AI Agent 是否建立了清晰、完整的自我认知？

| 评分 | 描述 |
|------|------|
| 0-20 | 无身份认知，仅基础模板 |
| 21-40 | 有基础身份信息（名字、角色） |
| 41-60 | 有完整身份文档（AGENTS.md） |
| 61-80 | 身份文档详细，包含规则、偏好 |
| 81-100 | 身份认知完整，持续进化（AGENTS.md/SOUL.md/PROFILE.md） |

**评估指标：**
- 身份文档完整性
- 规则清晰度
- 偏好明确度
- 持续更新频率

## 背景与目的

### 背景

AI Agent 系统正在快速发展，但缺乏标准化的能力评估方法。不同的实现（OpenClaw、CoPaw 等）各有优劣，导致：

1. 无法客观对比不同系统
2. 难以测量进步
3. 无法识别改进方向
4. 难以跨团队分享最佳实践

### 目的

ClawAF 旨在：

1. **标准化评价** - 提供通用的AI Agent能力评估框架
2. **支持对比** - 实现不同系统和团队间的客观对比
3. **引导进化** - 识别改进领域，提供可操作的建议
4. **分享知识** - 建立社区，分享最佳实践和经验

### 原则

1. **实用性优先** - 标准来自实际协作经验
2. **可量化** - 所有维度都有清晰的量化指标
3. **可扩展** - 框架可适应不同场景
4. **社区驱动** - 开源，鼓励社区贡献


对比多个团队成员：

```bash

```

## 结果应用

### 1. 自我改进

根据评估结果：
- **低于平均分的维度** → 优先改进
- **接近下一等级的维度** → 持续优化
- **已满分的维度** → 分享经验

### 2. 团队基准测试

- 识别团队优缺点
- 建立内部基准
- 跨成员分享最佳实践

### 3. 进化规划

根据评估差距生成改进路线图。
